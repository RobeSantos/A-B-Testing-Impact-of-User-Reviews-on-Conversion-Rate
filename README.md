Project Overview

This project analyzes whether removing user reviews from a product page increases the conversion rate in an e-commerce environment.

The experiment compares two versions of a webpage:

Version A (Control): Page with user reviews

Version B (Variant): Page without user reviews

The goal is to determine if the design change is statistically significant and worth implementing from a business perspective.

Business Context

Before running the experiment, two key metrics were defined:

Baseline Conversion Rate: Current performance of the page

Minimum Detectable Effect (MDE): The smallest improvement that justifies the cost of changing the design

The business team defined what level of improvement would be meaningful enough to implement the change.

✅ Conclusion

The experiment suggests that:

Removing user reviews from the product page leads to an increase in conversion rate.

Since the result is statistically significant and meets the business-defined MDE, the change can be considered for implementation.
